# aui

#### 简介
> aui是一套基于原生javascript开发的移动端UI库，包含常用js方法、字符校验、dialog提示弹窗、项目常用模板...
#### 特点
> 1、可以减轻APP、移动端网页开发工作量；</br>
> 2、调用方式简单、易懂；</br>
> 3、轻量级；</br>
#### 用法
1.loading加载动画
```javascript
aui.showload({msg: "加载中"});
```
