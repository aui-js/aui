# aui移动端UI框架

## 简介

> aui 是一套基于原生javascript开发的移动端UI组件库，包含常用js方法、字符校验、dialog提示弹窗、数字键盘、侧滑菜单、时间选择器、多级联动、聊天UI、项目常用模板......

## 特点

> 1、标准化，产品化</br>
> 2、更多复用，更快效率</br>
> 3、多人协作，更加统一</br>
> 4、维护方便</br>
> 5、易于使用</br>
> 6、减少UI设计时间
